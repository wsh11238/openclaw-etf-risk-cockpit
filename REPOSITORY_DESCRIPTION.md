# 仓库简介 / Repository Description

ETF 风控驾驶舱：一个纯前端 ETF 组合风控系统，把资产风险评分、组合体检、低估雷达、压力测试、回测入口、数据状态和周报预览集中到一个静态驾驶舱。适合学习金融工具、投研看板和高密度信息界面设计。

ETF risk cockpit: a static portfolio-risk dashboard for ETF research, including asset scorecards, portfolio review, undervaluation radar, stress testing, backtest entry points, data status, and report previews.
